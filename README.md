# 🔥 Firebase To-Do App

Bu proje, React ve Vite kullanılarak geliştirilmiş, arkasında veri tabanı ve kullanıcı yönetimi için Firebase barındıran gelişmiş bir yapılacaklar listesi (To-Do) uygulamasıdır. Vercel üzerinde canlıya alınmıştır.

## 🚀 Canlı Demo
Projenin canlı haline buradaki bağlantıdan ulaşabilirsiniz:  
👉 [todo-app-52nn.vercel.app](https://todo-app-52nn.vercel.app)

## ✨ Özellikler
* 🔐 **Google Auth:** Güvenli bir şekilde Google hesabınızla giriş yapabilirsiniz.
* 📝 **CRUD İşlemleri:** Görev ekleyebilir, tamamlandı olarak işaretleyebilir veya silebilirsiniz.
* 💾 **Realtime Database:** Eklediğiniz tüm veriler Firebase Firestore üzerinde anlık olarak saklanır.
* 🎨 **Modern Arayüz:** Kullanıcı dostu ve hızlı tepki veren şık tasarım.

## 🛠️ Kullanılan Teknolojiler
* **Frontend:** React, Vite, CSS3
* **Backend / Database:** Firebase Auth & Firestore
* **Deployment:** Vercel & GitHub
